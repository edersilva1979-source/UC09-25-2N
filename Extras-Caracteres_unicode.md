# 📖 O que é Unicode?

Unicode é um padrão internacional criado para representar caracteres utilizados no mundo inteiro.

Ele reúne letras, números, símbolos, emojis e caracteres especiais em um único conjunto.

Graças ao Unicode podemos escrever textos como estes:

```text
Olá!
こんにちは
مرحبا
Привет
💻📚🚀
```

Tudo isso faz parte do Unicode.

# 💡 Como Colocar um Emoji no `console.log()`?

É muito simples.

Basta escrever o emoji entre as aspas.

```typescript
console.log("💻 Computador");
```

Outro exemplo.

```typescript
console.log("📚 Biblioteca");
```

Ou.

```typescript
console.log("🚀 Sistema iniciado!");
```

Resultado.

```text
💻 Computador

📚 Biblioteca

🚀 Sistema iniciado!
```

# 🪟 Como Abrir a Lista de Emojis no Windows

O Windows possui um teclado de emojis integrado.

Pressione:

```text
Windows + .
```

ou

```text
Windows + ;
```

Aparecerá uma janela semelhante a esta.

```text
😀 😁 😂 🤖 💻 📚 🚀 ❤️ ⭐ 🎮
```

Agora basta clicar no emoji desejado.

Ele será inserido exatamente onde o cursor estiver.

---

# 🧪 Exemplo

Digite.

```typescript
console.log("");
```

Coloque o cursor entre as aspas.

Pressione.

```text
Windows + .
```

Escolha o emoji.

Resultado.

```typescript
console.log("🎮");
```

Muito simples.

# 🎨 Criando Menus Bonitos

Também podemos utilizar caracteres Unicode para desenhar menus.

```typescript
console.log("┌────────────────────────────┐");
console.log("│      MENU PRINCIPAL        │");
console.log("├────────────────────────────┤");
console.log("│ 1 - Cadastrar              │");
console.log("│ 2 - Consultar              │");
console.log("│ 3 - Relatórios             │");
console.log("│ 4 - Sair                   │");
console.log("└────────────────────────────┘");
```

Resultado.

```text
┌────────────────────────────┐
│      MENU PRINCIPAL        │
├────────────────────────────┤
│ 1 - Cadastrar              │
│ 2 - Consultar              │
│ 3 - Relatórios             │
│ 4 - Sair                   │
└────────────────────────────┘
```
