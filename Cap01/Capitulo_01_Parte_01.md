# Curiosidade

O Java foi apresentado ao mercado em 1995.

Seu principal objetivo era permitir que um programa pudesse ser
executado em diferentes sistemas operacionais sem precisar ser
reescrito.

Daí surgiu o famoso lema:

> **Escreva uma vez. Execute em qualquer lugar.**

------------------------------------------------------------------------

# Como um programa Java funciona?

``` text
Programador
      │
      ▼
Código Java (.java)
      │
      ▼
Compilador (javac)
      │
      ▼
Bytecode (.class)
      │
      ▼
JVM
      │
      ▼
Programa executando
```

O computador não entende Java diretamente.

Primeiro o código é compilado.

Depois a Máquina Virtual Java (JVM) executa esse código.
